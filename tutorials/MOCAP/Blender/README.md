## Part 1: 3D Art and Animation in Blender

Blender is a free and open-source 3D creation suite used by artists, designers, animators, and game developers. These beginner-friendly tutorials introduce the Blender interface, object manipulation, 3D modelling, materials, lighting, cameras, rigging, and animation.

The activities are designed to build foundational skills while encouraging creativity, experimentation, and play.

## What Is 3D Art?

**3D art** uses three-dimensional digital space to create objects with height, width, and depth. Unlike a flat image, a 3D object can be viewed from different angles, lit with virtual lights, placed inside an environment, and animated over time.

In this program, participants use 3D art to design original objects, characters, and environments that can later be animated using keyframes or recorded motion-capture data.
