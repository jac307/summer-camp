# Digital Arts Summer Camps Tutorials

Welcome to the **Digital Arts Summer Camps tutorial hub**. This website provides step-by-step guides for the software and creative tools used by children and teens participating in the summer arts programs at McMaster University.

Participants can use these tutorials during camp activities or revisit them independently to review techniques, complete projects, and continue experimenting after each session. The tutorials introduce tools such as **GIMP**, **Blender**, **DaVinci Resolve**, **Krita**, and other digital platforms used for image-making, animation, video editing, 3D modelling, and interactive media.

## About

This website supports hands-on learning in digital arts through clear and accessible instructions. Each tutorial breaks a creative process into manageable steps and includes examples, exercises, and technical guidance.

The goal is to help participants build confidence with digital tools while developing their own artistic ideas. Rather than focusing only on software skills, the activities encourage experimentation, problem-solving, storytelling, collaboration, and creative exploration.
